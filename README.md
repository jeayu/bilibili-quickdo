# bilibili-quickdo
b站H5播放器快速操作

## 功能
- 双击全屏
- ```+``` ```-```键调节播放速度
- ```f```键全屏
- ```w```键网页全屏
- ```p```键暂停/播放
- ```d```键开启/关闭弹幕
- 可以配置自动播放、全屏和关闭弹幕
